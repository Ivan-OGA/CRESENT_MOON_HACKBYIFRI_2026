# StudyTime – Hackathon IFRI 2026

## 📌 Présentation du projet

StudyTime est une application web conçue pour aider les étudiants à organiser, analyser et optimiser leur temps d’étude.

Elle agit comme un coach académique intelligent en générant un planning personnalisé et en proposant des quiz réguliers pour évaluer la progression.

Projet réalisé dans le cadre du Hackathon IFRI 2026  
Thème : « Intégration efficace du numérique dans l’apprentissage »

---

## 🎯 Problématique

Beaucoup d’étudiants rencontrent :

- Une mauvaise organisation du travail
- Une accumulation de matières non maîtrisées
- Un manque de suivi personnalisé
- Une absence d’auto-évaluation régulière
- Une difficulté à gérer efficacement leur temps

StudyTime propose une méthode structurée et mesurable pour améliorer la réussite académique.

---

## 🚀 Fonctionnalités principales

- Profil étudiant personnalisé
- Génération intelligente de planning
- Module de rattrapage
- Quiz automatiques (note sur 20)
- Suivi de progression
- Analyse des performances
- Interface à thème sombre

---
